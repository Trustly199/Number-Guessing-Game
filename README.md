# Number-Guessing-Game
A Python game where the computer picks a random number and the user has to guess it. The script uses a loop to give "higher" or "lower" hints after each attempt until the correct number is found.

Random Generation: The script uses the random module to pick a secret number between 1 and 100.

Game Loop: A while True loop keeps the game running until the correct guess is made.

Logic & Feedback: It uses if/elif/else statements to tell the user if their guess is too high, too low, or exactly right.

Error Handling: A try/except block prevents the program from crashing if the user types something other than a number.
